# Immigration Integrity and Due Process Act (IIDPA)

This repository contains the legislative packet for the **Immigration Integrity and Due Process Act (IIDPA)** — a citizen-drafted framework for restoring control, fairness, and constitutional due process in the U.S. immigration system.

## Contents


- **Executive_Summary.md** – High-level overview of the problem, goals, and core reforms.
- **One_Pager.md** – Single-page handout with key points and talking tracks.
- **IIDPA_Full_Bill_Text.md** – Full draft bill text, organized by titles and sections.
- **Simulated_CBO_Score.md** – Rough, good-faith fiscal and impact estimates.
- **LICENSE** – Reuse and contribution terms.

## How to Use This Repository

- **Legislative staff & policymakers:**  
  Start with the **Executive Summary** or **One Pager**, then review the **Full Bill Text** and **Simulated CBO Score** for detail.

- **Researchers, advocates, and citizens:**  
  Use this as a starting point for discussion, critique, and improvement. Suggestions and technical edits are welcome.
